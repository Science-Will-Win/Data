# AIGEN BioAgent - 7일 실행 결과

## 프로젝트 개요

AIGEN Sciences × Modulabs 협업 프로젝트로, 바이오메디컬 AI 에이전트를 개발·평가하는 것이 목표입니다.
150개 이상의 생물정보학 도구와 59개 생물학 데이터베이스를 활용하여 복잡한 바이오메디컬 연구 과제를 수행합니다.

- **실행 기간**: 2026.02.15 (1일차) ~ 2026.02.22 (8일차)
- **담당자**: 박정훈 (Hoony)
- **서버**: DGX A100 GPU 6-7 (211.206.241.242)
