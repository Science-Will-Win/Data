# 서버 접속 가이드

## 접속
```bash
ssh sww@211.206.241.242 -p 22
conda activate biomni-agent
cd ~/aigen-bioagent
```

## tmux 세션
```bash
tmux list-sessions        # 세션 목록
tmux attach -t work       # 세션 접속
# Ctrl+B → D              # 세션에서 나오기 (작업 유지)
```

| 세션명 | 용도 |
|--------|------|
| work | 메인 작업 |
| vllm1/vllm2 | vLLM 서버 |
| gdpo | GDPO 학습 (완료) |

## 모니터링
```bash
watch -n 30 bash ~/aigen-bioagent/mon_all.sh
```
